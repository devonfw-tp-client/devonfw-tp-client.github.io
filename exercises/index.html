Your first Angular (reactive) form (1/3)

  1. Create a new component, e.g. UserDetails: ng g component UserDetails.
  2. Go to user-details.component.html and add a <form> element.
  3. Add two input fields to the form: Name and Email. Make use of the Angular Material input
  4. Eventually add a submit button Send. Make use of the Angular Material button
  5. Don't forget to import necessary modules such as MatFormFieldModule, MatInputModule and MatButtonModule.

Your first Angular (reactive) form (2/3)

  1. Also, import ReactiveFormsModule.
  2. Go to user-details.component.ts and add a form property of type FormGroup.
  3. Inject a FormBuilder into the constructor and use it to create a form group containing the name and email controls. Assign the form group to the form property
  4. Add the send() method which prints out the current name's and email's value to the console.

Your first Angular (reactive) form (3/3)

  1. Go to user-details.component.html and put the [formGroup] property binding to the form property on the <form> element.
  2. Set the formControlName attributes on <input> elements. As values use the names which you picked for the a FormBuilder.
  3. Add the (submit) event binding to the send() method on the <form> element.
